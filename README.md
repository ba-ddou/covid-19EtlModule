# covid-19 ETLmodule

## Dependencies
To use this module you must have Node.js and GIT installed on your machine

## Data

```
> yarn load:data
```


## run the module

make sure you have node installed on your system

```
> yarn install
> yarn start
```

or

```
> npm install
> npm start
```

## Exports

The module exports 3 json file

-   dates.json
-   territories.json
-   stats.json

these files can be found in `./data/exports`
